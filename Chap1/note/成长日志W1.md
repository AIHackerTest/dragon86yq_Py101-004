- 初步掌握__name__的用法
    - __name__是一个系统定义的变量；
    - 如何直接运行test.py，__name__ == '__main__'；
    - 如何从另一个模块使用from...import导入test.py中main()，这时__name__== 'test'
    ```
    # test.py
    def main():
        print("we are in %s"%__name__)
        if __name__ == '__main__':
            main()
    >>> we are in __main__
    
    # another.py
    from test import main
    main()
    >>> we are in test
